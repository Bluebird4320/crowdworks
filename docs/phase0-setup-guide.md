## 🎯 全体像

### ゴール
- ✅ 動作する Googleスプレッドシート版を構築
- ✅ 10人のテストユーザーからフィードバック取得
- ✅ Webアプリ化への改善リストを作成

### 提供する主要機能
1. **ダッシュボード** - KPI 表示、おすすめ案件表示
2. **ジャンル別分析** - 相場、初心者向け度、難易度
3. **初心者OK案件一覧** - 安全な案件のみ表示
4. **スコアリング機能** - 100点満点で案件を評価
5. **地雷案件チェック** - AI キーワード検出
6. **応募文テンプレート** - 6種類のテンプレート
7. **月5万円シミュレーター** - 行動計画を自動生成

## 📊 各シートの詳細

### 📋 シート1：Input（入力用）

**役割**：案件データの自動受け取り

**設定方法**：
```
1. Googleフォームの回答をこのシートに自動送信
2. フォーム送信 → Input シートに自動記入

【カラム一覧】
A: タイムスタンプ（自動）
B: 案件ID（フォーム→自動）
C: 案件タイトル（フォーム→自動）
D: ジャンル（フォーム→自動）
E: 報酬（フォーム→自動）
F: 作業時間（フォーム→自動）
G: 発注者名（フォーム→自動）
H: 発注者評価（フォーム→自動）
I: 初心者OK（フォーム→自動）
J: 継続案件（フォーム→自動）
K: 危険キーワード（フォーム→自動）
L: リスク判定（自動計算）
M: スコア（自動計算）
N: 時給（自動計算）
```

**計算式**：
```
L列（リスク判定）：
=IF(N2<600, "高", IF(N2<800, "中", "低"))

M列（スコア計算）：
=IF(K2="",
  IF(N2>=1200, 25, IF(N2>=1000, 20, IF(N2>=800, 15, IF(N2>=600, 10, 0))))
  + IF(H2>=4.5, 20, IF(H2>=4, 17, IF(H2>=3.5, 14, IF(H2>=3, 11, 0))))
  + IF(I2=TRUE, 15, IF(J2=TRUE, 12, 0))
  + 15
  + IF(J2=TRUE, 10, 0)
  + 10
  + 5,
  IF(K2="外部誘導", 0,
    IF(N2>=1200, 25, IF(N2>=1000, 20, IF(N2>=800, 15, IF(N2>=600, 10, 0))))
    + IF(H2>=4.5, 20, IF(H2>=4, 17, IF(H2>=3.5, 14, IF(H2>=3, 11, 0))))
    + IF(I2=TRUE, 15, IF(J2=TRUE, 12, 0))
    + 0
    + IF(J2=TRUE, 10, 0)
    + 10
    + 5
  )
)

N列（時給計算）：
=IF(F2>0, E2/F2, 0)
```

### 📊 シート2：Jobs（案件一覧）

**役割**：すべての案件を一覧表示

**設定方法**：

```
1. Input シートのデータを参照
2. 自動的に同期される設定にする

【カラム一覧】
A: 案件タイトル
B: ジャンル
C: 報酬
D: 時給（Input から参照）
E: 初心者OK
F: 継続案件
G: リスク（Input から参照）
H: スコア（Input から参照）
I: 推奨度（スコアから自動判定）
J: アクション

【フォーミュラ】
I列（推奨度）：
=IF(H2>=85, "🌟 S ランク", IF(H2>=70, "🌟🌟 A ランク", IF(H2>=55, "🌟🌟🌟 B ランク", "△ 検討要")))

J列（アクション）：
「応募文を生成」ボタンへのリンク
```

**ソート機能**：
```
- データ > ソート範囲
- 時給が高い順（デフォルト）
- スコア順
- 報酬順
```

**フィルタ機能**：
```
- データ > フィルタを作成
- 初心者OK = TRUE
- リスク = 「低」のみ
- スコア >= 70
```

### 📈 シート3：Dashboard（ダッシュボード）

**役割**：全体の KPI とトレンドを表示

**レイアウト**：

```
【セクション1：KPI（4枚のカード）】
┌──────────────┬──────────────┬──────────────┬──────────────┐
│ 総案件数     │初心者OK      │ 平均報酬    │ 地雷リスク  │
│ =COUNTA()   │ =COUNTIF()   │ =AVERAGE()   │ 低:88%      │
│ ↑ 2%        │ ↑ 5%         │ ↑ 8%        │ 中:10%      │
└──────────────┴──────────────┴──────────────┴──────────────┘

【セクション2：グラフ】
- 折れ線グラフ：過去30日のジャンル別案件推移
  → 横軸：日付、縦軸：案件数
  → ジャンル別に色分け

- 円グラフ：初心者向け案件の割合
  → 「初心者OK」vs「初心者NG」

【セクション3：ランキング】
- おすすめジャンル TOP5
- 応募すべき案件 TOP5
- 避けた方がいい案件 TOP5
```

**計算式**：

```
総案件数：
=COUNTA(Jobs!A:A)-1

初心者OK案件数：
=COUNTIF(Jobs!E:E, TRUE)

平均報酬：
=AVERAGE(Jobs!C:C)

中央値報酬：
=MEDIAN(Jobs!C:C)

地雷リスク（低の割合）：
=COUNTIF(Jobs!G:G, "低") / COUNTA(Jobs!G:G)
```

### 📊 シート4：CategoryAnalysis（ジャンル分析）

**役割**：ジャンル別の詳細分析

**カラム一覧**：
```
A: ジャンル名
B: 案件数
C: 平均報酬
D: 中央値報酬
E: 初心者OK率（%）
F: 継続案件率（%）
G: リスク「高」率（%）
H: 難易度（1-5）
I: 初心者おすすめ度（⭐1-5）
J: 月の稼げる金額（目安）
```

**フォーミュラ例**：

```
【ライティングの場合】
B（案件数）：
=COUNTIF(Jobs!B:B, "ライティング")

C（平均報酬）：
=AVERAGEIF(Jobs!B:B, "ライティング", Jobs!C:C)

D（中央値報酬）：
=MEDIAN(IF(Jobs!B:B="ライティング", Jobs!C:C))

E（初心者OK率）：
=COUNTIFS(Jobs!B:B, "ライティング", Jobs!E:E, TRUE) / B / 100

F（継続案件率）：
=COUNTIFS(Jobs!B:B, "ライティング", Jobs!F:F, TRUE) / B / 100

I（初心者おすすめ度）：
=IF(E >= 0.7, IF(H <= 2, "⭐⭐⭐⭐⭐", "⭐⭐⭐⭐"), IF(E >= 0.5, "⭐⭐⭐", "⭐⭐"))

J（月の稼げる金額）：
=C * 8  # 月平均8件応募で成約した場合
```

### ⚠️ シート6：RiskCheck（地雷案件チェック）

**役割**：入力した案件の危険度を判定

**使い方**：

```
【手動入力エリア】
A1: 案件タイトル
A2: 案件説明

【自動判定結果】
C1: リスクスコア（0-100）
C2: リスク判定（低/中/高）
C3: 警告メッセージ
C4: 代替案（おすすめ案件へのリンク）
```

**危険キーワード判定**：

```
D列に危険キーワードリストを作成：
D1: "LINE登録"      → リスク：99
D2: "1記事100円"    → リスク：95
D3: "テスト案件"    → リスク：88
D4: "1件500円"      → リスク：80
...

E列：検出した危険キーワードをカウント
=SUMPRODUCT(--ISNUMBER(SEARCH(D:D, A1&" "&A2)))

F列：リスクスコア計算
=IF(E1 > 0, MIN(99, 50 + E1*10), IF(F1<600, 70, IF(F1<800, 50, 30)))
```

**結果の表示**：

```
C1: =F1
C2: =IF(C1 >= 66, "🔴 高", IF(C1 >= 31, "🟡 中", "🟢 低"))
C3: =IF(C1 >= 85, "⛔ 応募を強く推奨しません", IF(C1 >= 66, "⚠️ 事前確認してから応募", "✅ 安心して応募できます"))
```

### 📝 シート7：ApplyText（応募文生成）

**役割**：応募文のテンプレートと自動生成

**レイアウト**：

```
【セクション1：テンプレート選択】
A1: テンプレートタイプ（ドロップダウン）
- 初心者向け
- 丁寧で誠実
- AI活用アピール
- 継続案件向け
- 短文
- 実績少ない人向け

【セクション2：入力フォーム】
A3: ユーザー名（入力）
A4: スキル（複選）
A5: 経験数（入力）
A6: 案件タイトル（参照）

【セクション3：自動生成結果】
B3-B50: 応募文が自動生成
（テンプレート + 入力情報で埋め込み）

【セクション4：アクション】
B52: 📋 コピーボタン（スクリプト）
B53: ✏️ 手動編集エリア
```

**テンプレート例**：

```
【初心者向けテンプレート】
「初めまして。この度はご応募させていただきました。

私は現在「[スキル]」を習得中で、
これまで[経験数]件の案件を成約させてきました。

貴社の「[案件タイトル]」のお仕事に強い関心があり、
精一杯頑張らせていただきたいと思っています。

初心者のため至らない点も多いと思いますが、
ご指導いただければ幸いです。

どうぞよろしくお願いいたします。」

【AI活用アピールテンプレート】
「初めまして。この度はご応募させていただきました。

私は ChatGPT や Claude などの AI ツールを活用して、
「[スキル]」の案件を効率的に処理しています。

AI を活用することで、通常より 30-50% 短い時間で
高品質な成果物を納品することができます。

これまで[経験数]件の案件で AI 活用による
効率化で高い満足度をいただいています。

貴社の「[案件タイトル]」についても、
AI の力を最大限活用して対応させていただきます。

どうぞよろしくお願いいたします。」
```

### 📌 シート8：SavedJobs（ブックマーク管理）

**役割**：保存した案件の管理

**カラム**：
```
A: 保存日
B: 案件タイトル
C: ジャンル
D: 報酬
E: 応募状況（ドロップダウン）
   - 未応募
   - 応募済み
   - 採用
   - 不採用

G: アクション（応募ボタン）
H: 削除ボタン
```

**削除機能**：
```
Google Apps Script で実装：
function deleteRow(sheetName, rowNumber) {
  var sheet = SpreadsheetApp.getActiveSheet();
  if (sheet.getName() === sheetName) {
    sheet.deleteRow(rowNumber);
  }
}
```

## 💡 Tips

### 計算式が複雑な場合

```
→ Google Apps Script を使用
機能：
- 複雑な計算をカスタム関数で実装
- 例：スコアリング機能
- 例：危険キーワード検出
```

### グラフの表示がおかしい場合

```
→ データ > グラフエディタ
1. データ範囲を再確認
2. グラフの種類を確認（折れ線か円か）
3. 軸ラベルを確認
```

### テストユーザーを招待できない場合

```
→ 共有リンクを作成
1. 右上「共有」ボタン
2. 「全員に公開」をクリック
3. リンクをコピー
4. テストユーザーにメール送信
```
